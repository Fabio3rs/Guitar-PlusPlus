# BASS Audio Libraries

This directory should contain the BASS and BASS_FX libraries required for audio playback.

## Required Files

Download from [un4seen.com](http://www.un4seen.com/) and place here:

### Headers
- `bass.h`
- `bass_fx.h`

### Libraries
**Linux:**
- `libbass.so`
- `libbass_fx.so`

**Windows:**
- `bass.dll`
- `bass_fx.dll`

**macOS:**
- `libbass.dylib`
- `libbass_fx.dylib`
